# SL-GEM Data Editor

SL-GEMのゲームデータを編集するためのGUIツールです。Tauriを使用したデスクトップアプリケーションとして実装されています。

## 主な機能

- マップエディタ
  - マップの作成・編集
  - 地形設定
  - ユニット配置
- 勢力エディタ
  - 勢力データの編集
  - 初期配置設定
  - 関係性設定
- データ検証
  - バリデーションチェック
  - エラー表示
- データ管理
  - インポート/エクスポート
  - バージョン管理

## 技術スタック

- Tauri (デスクトップアプリフレームワーク)
- フロントエンド
  - TypeScript
  - React/Vue/Svelteなど
- バックエンド
  - Rust
  - sl-gem-model (共通ライブラリ)
